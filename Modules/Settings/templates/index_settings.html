Enable Indexed DB Source?

Enabling the Winget pre-indexed DB source will replace the default REST source. As a result:

  • The REST source will no longer function.
  • Client authentication will not be available.
  • IP restriction remains supported.
Disable Client Authentication?

Disabling client authentication for the REST source means that authentication will no longer be required to access it. As a result:

  • Any client within an allowed IP range will be able to download software without authentication.
  • Make sure your IP restrictions are configured correctly to limit unintended access.
